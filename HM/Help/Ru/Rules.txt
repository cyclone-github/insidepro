
#	комментарий
:	ничего не делать с исходным паролем
l	перевести в нижний регистр (PASSWORD -> password)
u	перевести в верхний регистр (password -> PASSWORD)
c	перевести первый символ в верхний регистр, остальные - в нижний (password -> Password)
C	перевести первый символ в нижний регистр, остальные - в верхний (password -> pASSWORD)
E	перевести в верхний регистр первый символ и каждый символ после пробела (my password -> My Password)
r	обратить (password -> drowssap)
d	дублировать (Password -> PasswordPassword)
f	отразить (Password -> PassworddrowssaP)
z	дублировать первый символ (password -> ppassword)
Z	дублировать последний символ (password -> passwordd)
k	обменять местами два первых символа (password -> apssword)
K	обменять местами два последних символа (password -> passwodr)
q	дублировать все символы (password -> ppaasswwoorrdd)
t	инвертировать регистр всех символов (PaSSword -> pAssWORD)
U	дублировать в верхнем регистре (password -> passwordPASSWORD)
{	ротация влево на один символ (password -> asswordp)
}	ротация вправо на один символ (password -> dpasswor)
[	удалить первый символ (password -> assword)
]	удалить последний символ (password -> passwor)
TN	инвертировать регистр символа в позиции N (T3 = password -> paSsword)
DN	удалить символ в позиции N (D2 = password -> pssword)
'N	обрезать до длины N символов ('6 = password -> passwo)
+N	инкремент символа в позиции N (+1 = password -> qassword)
-N	декремент символа в позиции N (-8 = password -> passworc)
yN	дублировать первые N символов, N = 2...9 (y3 = password -> paspassword)
YN	дублировать последние N символов, N = 2...9 (Y4 = password -> passwordword)
pN	копировать N раз, N = 2...9 (p3 = password -> passwordpasswordpassword)
$X	добавить в конец символ X ($3 = password -> password3)
^X	добавить в начало символ X (^5 = password -> 5password)
@X	удалить все символы X (@s = password -> paword)
iNX	вставить в позицию N символ X (i2Q = password -> pQassword)
oNX	заменить символ в позиции N на символ X (o4Z = password -> pasZword)
*NM	обменять местами символы в позициях N и M (*23 = password -> psasword)
sXY	заменить все символы X на символы Y (ss3 = password -> pa33word)
SXY	заменить первое вхождение символа X на символ Y (Ss$ = password -> pa$sword)

Примечание:
	N = "1...9" для позиций 1...9, "A...Z" для позиций 10...35, "a...z" для позиций 36...61
