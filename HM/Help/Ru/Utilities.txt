
*******************
 Hash List Manager
*******************

1. После запуска программы нужно нажать F2 и выбрать каталог со списками хэшей.

2. Файлы в этом каталоге должны именоваться в следующем формате:
	"xxx.N.txt", где N - тип хэшей из программы hashcat

Примеры:
	"MyMD5.0.txt" для хэшей MD5;
	"Uncracked.100.txt" для хэшей SHA-1;
	"Left(MyBB).2811.txt" для хэшей MyBB;
	"Contest_hashes.3200.txt" для хэшей Blowfish(OpenBSD);
	и т.д.

3. В том же каталоге обязательно должен присутствовать файл "~All.pot" с паролями
для этих хэшей (в формате "хэш:пароль" или "хэш:соль:пароль").

4. После выбора каталога (или по нажатию F5) программа сгенерирует статистику по всем
загруженным спискам и создаст в подкаталоге \Temp следующие рабочие файлы:
	- остатки по каждому типу хэшей (~N.txt);
	- остатки по каждому списку хэшей (*.txt);
	- словарь для каждого списка хэшей (*.dic);
	- файл с паролями для каждого списка (*.pot);
	- общий файл с паролями (~All.pot);
	- общий словарь (~All.dic).
